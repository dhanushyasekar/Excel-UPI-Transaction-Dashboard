# Excel UPI Transaction Dashboard 📊

## Project Overview

This project is an interactive Excel dashboard created to analyze UPI transaction data and generate business insights.

The dashboard helps understand revenue, expenses, profit, payment methods, departments, and regional transaction performance.

## Tools Used

- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Data Analysis
- Dashboard Design


## Key Analysis

- Profit analysis by payment method
- Revenue analysis by category
- Department-wise revenue, expenses and profit comparison
- Regional transaction analysis
- Average expense analysis
